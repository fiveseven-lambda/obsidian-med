偽性低 Na 血症を理解するためには、まず「血漿浸透圧を左右するのは**血漿から脂質とタンパク質を除いた水分の $\ce{Na+}$ 濃度**である」という点を理解する必要がある。

|                    | 血液  | 血漿  | 血清  |                    |
| ------------------ | --- | --- | --- | ------------------ |
| 水分                 | ○   | ○   | ○   | ←ここの $\ce{Na+}$ 濃度 |
| 凝固因子以外のタンパク質<br>脂質 | ○   | ○   | ○   |                    |
| 凝固因子               | ○   | ○   | ☓   |                    |
| 血球                 | ○   | ☓   | ☓   |                    |

これに対し、生化学検査では「希釈した血清の $\ce{Na+}$ 濃度」を測定している。このとき、凝固因子以外のタンパク質・脂質が多いと、水分が希釈されすぎるので、たとえホメオスタシスが正常でも検査結果は $\ce{Na+}$ 低値になってしまう。具体的な原因として脂質異常症や多発性骨髄腫などがある。

鑑別には血清浸透圧を測定する。血清浸透圧が正常なら、ホメオスタシスは保たれており、真の低 $\ce{Na+}$ 血症ではないといえる。ここで **$\ce{Na+}$・尿素窒素・血糖値を用いた浸透圧の推定値を用いてはいけない**。$\ce{Na+}$ の検査値が正しいか知りたいのに、$\ce{Na+}$ を使って推定するのはおかしい。

また、血液ガス分析は、希釈しないためこれらの影響を受けない。
